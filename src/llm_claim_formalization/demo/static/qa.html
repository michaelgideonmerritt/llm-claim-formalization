🔍 Verified Question-Answering

LLM knowledge + Formal verification + Feedback loop = Deterministic, hallucination-free answers

Claim Verification Question Answering

How It Works

  1. User asks question
  2. Verifier classifies question type (arithmetic, logic, factual)
  3. LLM generates answer using training knowledge
  4. Verifier checks if answer is deterministic and logical
  5. If verified ✅ → Send to user
  6. If rejected ❌ → Send feedback to LLM → Retry
  7. Loop until verified or max retries
